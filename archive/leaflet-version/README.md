# Leaflet Map Version Archive

**Archived Date:** November 9, 2025

## Description

This folder contains the archived version of the Sports Betting Dashboard that used Leaflet.js for the interactive map visualization.

## Files Included

- `index.html` - Main HTML file with Leaflet map implementation
- `main.js` - JavaScript file with map functionality

## Leaflet Implementation Details

This version used:
- Leaflet.js v1.9.4 for interactive mapping
- Custom state polygons and choropleth visualization
- Interactive tooltips and state information display

## Reason for Archive

Archived to preserve the Leaflet map implementation before transitioning to a different mapping solution or making significant changes to the codebase.

## How to Use

To view this archived version:
1. Open `index.html` in a web browser
2. The map will load with all original Leaflet functionality
