# Sports Betting Policy Dashboard - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, trustworthy foundation
- **Secondary**: Warm Copper (#c17a5b) - Data highlights and interactive elements  
- **Accent**: Sage Green (#8fa68e) - Success states and positive metrics
- **Neutral**: Light Gray (#f8f9fa) - Background and subtle elements
- **Text**: Charcoal (#2d3748) - High contrast readability

### Typography
- **Display Font**: Tiempos Headline - Bold, authoritative for headers and key metrics
- **Body Font**: Suisse Int'l - Clean, readable for data and content
- **Data Font**: SF Mono - Monospace for precise numerical displays

### Visual Language
- **Editorial Sophistication**: Inspired by premium publications like Wired and The Gentlewoman
- **Data-Driven Aesthetics**: Clean, precise visualizations with purposeful color usage
- **Professional Authority**: Conveying expertise and reliability in policy analysis
- **Subtle Depth**: Layered information architecture without overwhelming complexity

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth timeline animations and state transitions
- **ECharts.js**: Interactive data visualizations and charts
- **p5.js**: Generative background patterns and creative coding elements
- **Splitting.js**: Advanced text animation effects for headers
- **Typed.js**: Dynamic text reveals for key insights

### Animation & Effects
- **Timeline Progression**: Animated state-by-state legalization sequence
- **Data Reveals**: Staggered chart animations with smooth easing
- **Hover States**: Subtle 3D transforms on interactive elements
- **Background**: Subtle geometric patterns using p5.js
- **Text Effects**: Split-letter animations for major headings

### Header Effect
- **Generative Background**: Animated geometric grid pattern
- **Gradient Overlay**: Subtle depth without overwhelming content
- **Parallax Elements**: Layered visual hierarchy
- **Dynamic Typography**: Animated text reveals with color cycling

### Layout Principles
- **Grid System**: 12-column responsive layout with consistent spacing
- **Information Hierarchy**: Clear visual weight distribution
- **White Space**: Generous padding for premium feel
- **Content Flow**: Logical progression from overview to detailed analysis

### Interactive Elements
- **State Map**: Color-coded with hover details and click interactions
- **Timeline Scrubber**: Drag-to-explore legalization progression
- **Filter Controls**: Clean, accessible interface elements
- **Data Tables**: Sortable, responsive design with clear visual feedback

### Mobile Considerations
- **Touch-Friendly**: Minimum 44px touch targets
- **Simplified Navigation**: Collapsible menu system
- **Optimized Charts**: Mobile-specific chart configurations
- **Performance**: Reduced animations on mobile devices

## Content Strategy
- **Scannable Structure**: Clear headings and visual breaks
- **Progressive Disclosure**: Layered information access
- **Data Storytelling**: Narrative flow through visualizations
- **Professional Tone**: Authoritative yet accessible language
